## component Modal npm package

### Install

Install using `npm install @polettiste/modalcomponent`

### Usage

In a React app, use the Button component:  
`import { Modal } from '@polettiste/modalcomponent'

#### Button props

| Name     | Description |          |         |
| -------- | ----------- | -------- | ------- |
| title    | string      |          |         |
| text     | string      |          |         |
| data-id  | string      |          |         |
| type     | 'button'    | 'submit' | 'reset' |
| name     | string      |          |         |
| value    | string      |          |         |
| disabled | Boolean     |          |         |
| onClick  | function    |          |         |
